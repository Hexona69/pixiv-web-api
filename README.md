## Usage

`git clone`

`npm install`

## `/ranklist`

- type: `GET`

### query strings

- `offset`: What number the ranklist starts at
- `time`: Time period for the ranklist, the value could be 
  - `DAY`
  - `WEEK`
  - `MONTH`
  - `DAY_MALE`
  - `DAY_FEMALE`
  - `WEEK_ORIGINAL`
  - `WEEK_ROOKIE`
  - `DAY_MANGA`
